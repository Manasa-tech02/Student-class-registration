STUDENT REGISTRATION APP — PROJECT MAP

Stack
- Expo SDK 54 + React Native 0.81
- Expo Router (file-based routing) + typed routes enabled
- TypeScript (strict)
 - Node.js backend (Express + SQLite + JWT)

Routing (Expo Router)
- app/_layout.tsx
  - Root stack navigator
  - Routes:
    - (tabs) (header hidden)
    - modal (presentation: modal)

- app/(tabs)/_layout.tsx
  - Bottom tabs navigator
  - Tabs:
    - Home  -> app/(tabs)/index.tsx
    - Explore -> app/(tabs)/explore.tsx
    - Students -> app/(tabs)/students.tsx (renders feature screen from src/)

- app/modal.tsx
  - Modal route

Scalable code organization
- app/
  - ONLY routing + navigation wrappers (Expo Router)

- src/
  - Non-route app code organized by domain and reuse
  - features/
    - students/
      - types.ts (Student)
      - data/mockStudents.ts
      - screens/StudentListScreen.tsx
    - auth/
      - components/AuthLayout.tsx, AuthTextField.tsx
      - screens/LoginScreen.tsx, SignUpScreen.tsx
  - shared/ (planned): reusable UI + utilities (no business rules)
  - services/ (planned): integrations (storage, analytics, notifications)
  - lib/ (planned): low-level clients/wrappers
  - state/ (planned): app-level state (when needed)

Shared UI (template)
- components/
  - themed-text.tsx, themed-view.tsx, parallax-scroll-view.tsx, external-link.tsx, hello-wave.tsx
  - ui/: collapsible.tsx, icon-symbol*.tsx, haptic-tab.tsx

Hooks + theme
- hooks/
  - use-color-scheme.ts (+ .web.ts)
  - use-theme-color.ts
- constants/theme.ts

Assets + config
- assets/images/ (icons/splash/favicons referenced by app.json)
- app.json (Expo config, expo-router plugin, typedRoutes enabled)

Backend
- backend/
  - package.json (Express, SQLite, JWT, bcrypt)
  - src/db.js (SQLite database + migrations)
  - src/authRoutes.js (register, login, refresh, logout + audit logging)
  - src/server.js (Express app, CORS, JSON body parsing)

Import conventions
- TS alias: @/* -> project root
- Prefer for new code: "@/src/..." (example: app/(tabs)/students.tsx)
- Existing template imports: "@/components/...", "@/hooks/...", "@/constants/..."

Next feature ideas (optional)
- Students: add create/edit form, validation, persistence (AsyncStorage / SQLite), search, detail screen
- Registration: courses, enrollments, filters, offline-first sync
